# OpenCode NeMo-Flow Smoke Sample Outputs

These files are illustrative outputs from the one-task OpenCode NeMo-Flow Harbor
smoke workflow. They are intended as a quick shape reference for teammates
running the smoke locally.

The samples are not golden outputs. Model behavior, tool counts, and LLM judge
scores can drift between runs.

The CSV artifact uses normalized example paths instead of a local `.tmp` trial
directory.

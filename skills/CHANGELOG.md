# Changelog

All notable changes to the NeMo Agent toolkit AI coding agent skills are documented in this file.

The skills use a shared version because they are intended to work together as one routed skill set from `AGENTS.md`.

## [0.1.0] - 2026-05-12

### Added

- Added cuOpt-style root routing in `AGENTS.md`.
- Added focused skill folders for installation, workflow creation, agent configuration, tools and functions, evaluation, optimization, telemetry, MCP and serving, user rules, and skill evolution.
- Split Pascal repository source material into task-specific skill folders with one `SKILL.md` per folder.
- Added install documentation and example prompts for coding-agent skills.
